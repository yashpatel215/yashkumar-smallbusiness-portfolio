# Yashkumar Patel Portfolio Website

Static portfolio website for Yashkumar Patel's local visibility support service.

## Folder Structure

```text
Side Hustle/
├── index.html
├── styles.css
├── script.js
├── assets/
│   └── hero-workspace.png
├── README.md
└── GITHUB_PAGES_GUIDE.md
```

## Files

- `index.html` - Main website page and content.
- `styles.css` - Website layout, colors, responsive design, and mobile styling.
- `script.js` - Mobile menu behavior and email form draft behavior.
- `assets/hero-workspace.png` - Hero image used on the homepage.

## Local Preview

From this folder, run:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

## Publish

Follow [GITHUB_PAGES_GUIDE.md](GITHUB_PAGES_GUIDE.md) to publish this website for free with GitHub Pages.
